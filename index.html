Contribuer en tant que relecteur

Vous pouvez lire les leçons en cours de relecture et faire vos remarques en ouvrant une issue sur GitHub.
(vous pouvez ouvrir une seule issue pour toutes vos remarques sur une même leçon).

Contribuer en tant que collaborateur

Si vous êtes un collaborateur du projet. Pour un changement :

  • mineur (genre typo): effectuer directement la modification et l'indiquer dans les commentaires lors du commit.
  • court : ouvrir une issue pour décrire le changement voulu. Si c'est bien accueilli par le groupe, effectuer le changement en indiquant le numéro d'issue dans les commentaires du commit
  • conséquent : ouvrir une issue pour décrire le projet. Si c'est bien accueilli par le groupe, effectuer le changement dans une branche séparée puis (... à définir : pull request ?)
